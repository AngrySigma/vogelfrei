# Warrior class kit

Finishes a Warrior after the core `character-generation` workflow. Works on
the same character JSON (`--file`); `$S` =
`.claude/skills/character-generation/scripts`.

Stop if the JSON's `class` is not `Warrior`.

## 1. Verify the class grant

Level-1 Warriors start with **WS +1 and BS +1** (class table,
docs/Character/Classes/Warrior/index.md). `apply_class.py` sets this; check
`ws`/`bs` in the JSON and fix with
`python3 $S/annotate.py --file ... --set ws=1 --set bs=1` if they are off.

## 2. Record the advancement restriction

From level 2 on, Warriors gain +1 Combat Skill Point per level, spendable
only as the career allows (`career_combat_skills` in the JSON — "WS" or
"WS or BS"). Record it so it survives to the table:

```
python3 $S/annotate.py --file ... --note "Advancement: +1 Combat Skill Point/level, spendable on <WS | WS or BS> only (career)"
```

## 3. Apply the career trait

The trait text is already in the JSON (`career_trait`) and prints on the
sheet. What needs doing is its *mechanical* setup per career — read the
career page (`career_page`) and act:

| Career | Setup |
| --- | --- |
| Duellist | Fights with a weapon in each hand (one Minor/Small), rolling both damage dice, keeping higher. Make sure the inventory actually contains a suitable off-hand weapon (e.g. `buy.py add "Weapon (Minor)"`); note that a shield forfeits the trait. |
| Knight | Gains a 0-level squire henchman; henchmen get +half level (round up) to Loyalty and Morale. Add a note naming the squire (roll or invent a name with `roll_name.py`). |
| Mercenary | Weapon-switch trait needs no setup, but it rewards carrying a loaded backup weapon — suggest one if the purse allows. |
| Pit Fighter | Unarmed attacks deal d4; adds Strength AND Agility bonus to Grappling, wins ties. Add a note; a Cestus is a cheap thematic buy. |
| Soldier | Reduces Encumbrance by 1: `annotate.py --set enc_adjust=-1`, then **re-run finalize.py** so movement updates. Note the half forced-march damage and −1h sleep. |
| Witch Hunter | Adds +half level (round up, so +1 now) to Magic and Device saving throw rolls. Note it next to the saves — do not edit the save table itself. |

## 4. Sanity-check the loadout

- A Warrior with WS +1 can use Trained melee weapons (swords, polearms,
  great weapons); BS +1 opens all ranged weapons and firearms. If
  `finalize.py` warned about an unusable weapon on a Warrior, something is
  wrong — investigate.
- Careers restricted to "WS" still start with BS +1; a backup ranged
  weapon is rarely wasted.

## 5. Re-render

```
python3 $S/finalize.py --file ...        # only if equipment or enc_adjust changed
python3 $S/render_sheet.py --file ... --out characters/<slug>.md
```

Report what the trait means in play in one sentence when presenting the
character.

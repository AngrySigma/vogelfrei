# Thief

**Class**: [Rogue](../Rogue.md)

**Status**: Brass

**Skills**: Architecture, Bushcraft, Climb, Doors, Languages, Search, Sleight of Hand, Stealth, Tinkering

!!! tip "Trait"
    **Sneak Attack** — when a Thief strikes a target that is unaware of them or otherwise Surprised (as when they attack straight out of [Stealth](../../../../Adventuring/Skills.md)), they roll the weapon's damage dice twice and add both results together. A Thief is also the trained specialist the skill rules allow for: they may use the Climb skill even while encumbered.

| Level | Progression                  |
| ----- | ---------------------------- |
| 1     | 2 Skill Points, Sneak Attack |
| 2     | +2 Skill Points              |
| 3     | +2 Skill Points              |
| 4     | +2 Skill Points              |
| 5     | +2 Skill Points              |
| 6     | +2 Skill Points              |

The Thief starts with the base 1-in-6 chance every character has in the skills
above and spends Skill Points to improve them: each point raises one skill's
chance by 1-in-6. See [Skills](../../../../Adventuring/Skills.md) for what each
skill covers and how it is rolled.

# Character Creation

Character creation follows these steps in order:

1. Roll for [Ability Scores](Ability%20Scores.md)
    1. Note bonuses
    2. Is the character suitable?

2. Roll for [Class and Career](Class%20and%20Career.md)
    1. Determine Wounds
    2. Determine Weapon Skill, Ballistic Skill and Saving Throws
    3. Note Career Talents and Bonuses
    4. Note character Status
    - See [Classes](Classes/index.md) for all available classes and their careers

3. Select [Alignment](Alignment.md) or pick one determined by your Class and Career

4. Determine [Starting Possessions](Starting%20Possessions.md)
    1. Roll Money
    2. Buy Equipment — see [Equipment](../Equipment/index.md) for full lists

5. Choose a [Name](Name.md)

Once in play, see [Advancement](Advancement.md) for how characters improve over time.
